## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/Wasay-514/wasay-data-portfolio.git
```

Navigate into the project:

```bash
cd wasay-data-portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
