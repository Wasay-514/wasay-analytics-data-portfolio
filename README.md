# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Wasay-514/wasay-data-portfolio.git
```

Go to the project folder

```bash
cd wasay-data-portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Production build

```bash
npm run build
```

---

# 🌐 Deployment

This portfolio is deployed on **Vercel**.

```bash
npm run build
```

Output Folder

```text
dist
```
